#MCC_Ban_History

Description:
    This is a mod that allows you to look up the punishment history of any existing player on the MCCentral Server.
Instructions:
    In game, use one of the aliases followed by a player name.
    Ex. /history Gracefell
        /banHistory Gracefell
        /muteHistory Gracefell
        /his Gracefell
Downloading the Mod:
    1. Download the repository as a zip.
    2. Unzip the file.
    3. Add the MCC_Ban_History-1.0.jar file to your mods folder.
